# 🧑‍🦽 보건 동행 서비스 (Multi)
* KHIDI 공공데이터 활용 공모전


## 📌 프로젝트 개요
교통약자가 병원·보건 시설 등 이동이 필요한 상황에서 자원봉사자와 매칭되어
안전하게 동행 서비스를 받을 수 있도록 지원하는 웹 서비스입니다.

교통약자는 동행 공고를 등록하고,
자원봉사자는 공고를 확인 후 동행을 수락하여 매칭이 이루어집니다.
또한 교통약자를 위한 관련 제품 정보 및 보건 동행 서비스 시설 정보를 함께 제공합니다.

## 👥 팀 구성 및 역할
총 인원: 3명
풀스택 개발자 2명
데이터 수집 1명
담당 역할:
풀스택 개발
동행 매칭 기능 구현
게시글 CRUD 및 사용자 기능 개발


## ⚙️ 주요 개발 기능
동행 공고 게시 및 수정/삭제
동행 요청 수락 기반 매칭 기능
동행자 정보 확인 기능
교통약자 관련 제품 리스트 제공 (외부 URL 연동)
교통약자 보건 동행 서비스 시설 위치 정보 제공
로그인 및 회원가입 기능
